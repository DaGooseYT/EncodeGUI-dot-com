Subtitle Codecs


Encoding/muxing subtitles is supported in EncodeGUI. Below is a chart displaying the compatibility of each supported subtitle codec to each output container.


Codecs / Containers .mp4 .mkv .mov .3gp .webm
SubRip ✓
WebVTT ✓ ✓
ASS / SSA ✓
TTXT (Mov_Text) ✓ ✓ ✓
All of these codecs have been tested to be compatible with the
marked containers.


Enabling and Configuring Subtitles

The highlighted section in the photo below is where you set the state of the subtitles. Unckeck the Subtitles checkbox to disable subtitles in the output.

Select Auto encode in the drop-down menu to allow EncodeGUI to automatically convert the audio subtitles to the supported codec on the basis of the selected output container.


Note: Picture based subtitles are currently not supported in EncodeGUI. Please use text-based subtitles only to prevent errors.
