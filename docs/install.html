Installation Guide

This section will cover the installation for EncodeGUI and FrameGUI. FrameGUI v2.2.0 and newer is supported on Windows OS (Windows 10 newer) and macOS. As of v1.1.7, EncodeGUI is only supported on Windows OS (Windows 10 or newer). No Linux distro is currently supported yet.
This guide assumes that you have already downloaded either the EncodeGUI or FrameGUI setup file on the downloads page.



Accept the License Agreement

Please read the contents of the license agreement and click the I accept the agreement bubble to continue the setup process. A copy of this license is available here. Click the Next button.



Creating a Desktop Shortcut

Next, you'll want to decide whether or not you want an EncodeGUI shortcut icon on your desktop. Check or uncheck the checkbox and then click the Next button.



Confirm Installation

This page simply confirms the installation details. Click the Install button to continue.




Download Dependencies (If Applicable)

This page displays the download information for the additional required dependencies in EncodeGUI. Please wait while this process finishes. The speed of the download largely depends on your internet speeds and the current server load. If you do not see this page, then your system already has the required dependencies for EncodeGUI.

Note: If you get an error while attempting to download the dependencies, please wait and try again a few hours later and check your local network configuration. After that, if the error still persists, please contact DaGoose.



Installing the Downloaded Dependencies

This page will appear after the download finishes. Please wait as the contents that were just downloaded are being extracted and installed to the EncodeGUI directory. This process could take several minutes so please be patient. Once again, if you do not see this page, then your system already has the required dependencies for EncodeGUI.



Finishing Up

This page will appear once the EncodeGUI installation has completed. Click the Finish button and, if specified, EncodeGUI will open automatically. The installation is now complete.
