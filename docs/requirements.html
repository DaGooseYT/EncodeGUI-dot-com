System Requirements

A Heads-up for New Users:


Although EncodeGUI is easy to use, it’s not for everyone. If you are unfamiliar with most of the terms below, then EncodeGUI is most likely NOT for you:


• Video/Audio codec • CUDA • NCNN • Vulkan • AVC • HEVC • ProRes • Theora • VP9 • NVENC • QuickSync • AMF • Video container • Video resolution • Video profile • Video profile level • DeNoise • Frame interpolation • Artefacts • Muxing • Bitrate • CRF / Constant Rate Factor • Constant Quantizer • Rate control • Frame rate • Color depth • Color sampling • Color space • Color matrix • Color transfer • Color primaries • Color range


FrameGUI does not require knowledge relating to the terms above. However, you still must have basic computer skills, such as being able to copy files and compressing files into a .zip archive. This is necessary for when you need to report a problem to the developer.

Minimum PC Specifications (Windows):


In order to use all features in EncodeGUI or FrameGUI, your system must meet or exceed the minimum specifications listed below. AMD GPUs are supported but CUDA AI filters will not be supported. Consequently, AMD GPUs are generally not recommended for EncodeGUI. You can still use EncodeGUI or FrameGUI if your system doesn’t meet the minimum requirements, but AI filters will not be usable.


CPU GPU Storage Type RAM Screen Resolution OS
x64, 2-cores Nvidia GT 1030 SSD 8GB 1280x720 Windows 10
Any specs newer than what's listed above are assumed supported. ARM CPUs are not supported.
Note: If using an Nvidia GPU, it MUST be Pascal architecture or newer. Nvidia Maxwell GPUs or older are not supported for the AI filters. Intel integrated GPUs also aren't supported (with the exception of Xe).

Recommended PC Specifications (Windows):


CPU GPU Storage Type RAM Screen Resolution OS
i7-12700K RTX 3060 SSD 16GB 1920x1080 Windows 11
Nvidia RTX GPUs are recommended due to the presence of AI tensor cores.
Note: Upscaling/Interpolating 8K videos with the AI filters will need at least 24GB of system RAM. 32GB system RAM is necessary for the dual gpu processing feature.

Apple Macintosh Minimum Specifications (macOS):


CPU/GPU RAM Screen Resolution OS
Apple M1 8GB 1920x1080 Ventura 13.1
Any specs newer than what's listed above are assumed supported.

Note: FrameGUI for macOS is only supported on Apple Silicon Macs. We do not and will not provide support for Intel Macs or older versions of macOS.
