Video Codecs

This section will cover the available video codecs in EncodeGUI and how to use them.


General Info

With EncodeGUI, you have the option to use one of 6 video codecs for encoding, or muxing for video passthrough. Below is a chart displaying the compatibility of each supported video codec to each output container:


Codecs / Containers .mp4 .mkv .mov .avi .asf .ts .flv .3gp .ogv .webm
AVC ✓ ✓ ✓ ✓ ✓ ✓ ✓ ✓
HEVC ✓ ✓ ✓ ✓
ProRes ✓ ✓ ✓
Theora ✓ ✓
VP9 ✓ ✓ ✓ ✓ ✓
AV1 ✓ ✓ ✓ ✓ ✓
VP8 (Muxing only) ✓ ✓ ✓ ✓ ✓
VC1 (Muxing only) ✓ ✓ ✓ ✓
MPEG2 (Muxing only) ✓ ✓ ✓ ✓ ✓
MPEG4 (Muxing Only) ✓ ✓ ✓ ✓
All of these codecs have been tested to be compatible with the marked containers.
